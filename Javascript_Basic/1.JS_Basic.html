변수의 명령 규칙

        1. 영어 대/소문자, 숫자. _, $ 가용가능
        2. 첫글자 숫자 사용 불가
        3. 공백을 포함한 특수문자 사용불가
        4. 이름에 의미있는 단어의 조합 권장
        5. 예약어 사용 불가

        9. 변수, 인스턴스, 함수, 메소드는 항상 소문자로 시작

자료형

자바스크립트의 자료형은 숫자, 문자열, 논리형과 같은 기본 유형과
배열, 객체를 다루는 복합유형 그리고, undefined, null과 같은 특수유형이 있다.

숫자형


정수와 실수형이 있다.
정수는 소수점이 없는 숫자이다. 실수는 소수점이 있는 숫자이다.
ex. 3, 4, 5, 2023, 1992, ... ex. 3.1, 4.13, 5.18, 12.25, ...

문자열은 작은 따옴표나 큰따옴표로 묶는 데이터를 의미한다.
ex. “Hello Javascript”, 'Hello Javascript'

논리형

논리형은 불린유형이라고도 하며, 참이나 거짓의 값을 표현한다. ex. ture, false

undefined 와 nell

undefined는 대입연산자를 통해 값이 대입되지 않은 상태이고
null은 대입연산자를 통해 값이 대입되었으나 유효하지 않은 값이다.

연산자의 종류

1. 최우선 연산자 : (), [], .
2. 단항 연산자 : ++, --
3. 산술 연산자 : +, -, *, /, %
4. 관계비교 연산자 : >, <, >=, <=, ==, !=, ===, !==
5. 논리 연산자 : &&, ||
6. 대입 연산자 : =
7. 복합대입 연산자 : +=, -=, *=, /==, %=
8. 삼항 연산자 : 조건식 ? 참 : 거짓

제어문

1. 조건문 : if, if ~ else, else if, switch ~ case
2. 반복문 : for, while, do ~ while, for in
3. 분기문 : continue, break
